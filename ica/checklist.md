# Checklist: Predio Exportador

Si eres un ganadero en el Cesar y quieres certificar tu finca para exportación, debes cumplir con los siguientes puntos:

| Requisito | Estado |
| :--- | :---: |
| Registro Sanitario de Predio Pecuario (RSPP) | [ ] |
| 100% de animales con "chapa" (DIN) | [ ] |
| Certificado de Predio Libre de Brucelosis | [ ] |
| Certificado de Predio Libre de Tuberculosis | [ ] |
| Ciclos de Vacunación de Aftosa al día | [ ] |
| Bodega de insumos con llave | [ ] |
| Registro de tiempos de retiro de medicamentos | [ ] |

**Nota:** La visita técnica del ICA es indispensable para validar estos puntos en sitio.
